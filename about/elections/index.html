Elections

This page contains information on the upcoming elections for DormCon's 2022-2023 executive team.

About​

Are you interested in leading efforts to improve student dorm life? Want to join a chill student government? Run for a Dormcon Exec position! In addition to being responsible for a specific area of focus, all members of Dormcon Exec are involved in the discussions we have about student life issues and policies, and give input on the policies we advocate for.

Elections will happen on Thursday, May 5 at 6:30-8pm in the UA Chambers (W20-400).

Positions you can run for​

  • President
  • Vice President
  • Secretary
  • Treasurer
  • Housing Chair (2)
  • Dining Chair (2)
  • Upperclassmen Reps (2)
  • JudComm
  • 🌎 Tech Chair

Position descriptions can be found here. Regardless of your position on DormCon exec, you’ll be part of the conversation on all of the issues and projects we work on.

Interested in running?​

All non-graduating dorm residents are eligible to run for a position on Dormcon Exec (no experience necessary)! Feel free to reach out to dormcon-exec@mit.edu or any of the listed contacts for each position if you have any questions! Fill out this (non-binding) form to indicate your interest in a position.

If you’re interested in a role (or Dormcon in general) you can check us out at our next Dormcon GBM, which will be on Thursday, April 21 at 6:30-8pm at McCormick. Join the dormcon-announce mailing list for updates, meeting agenda, and reminders.

How to run for a position​

If you’d like to actually run for a position, you should fill out the intent to run form.1 Answers to this form will be shared at the election meeting, so keep in mind that these answers will be public.

If you’re not sure if you are going to run for a position, please fill out the form anyway - you can decide later that you’d rather not run!

Eligibility to run​

You’re eligible to run for DormCon exec if you’re planning to live in a dorm next year, with the exception of officers of the UA, IFC, Panhel, or LGC. No prior experience is required.

If you have any questions, please contact us at dormcon-exec@mit.edu!

  1. The intent to run form will be posted as soon as it is available.↩
